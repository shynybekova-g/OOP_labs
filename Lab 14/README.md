# Lab 14 — Packaging, Installing, and Managing Python Packages

## Objective
Бұл лаборатория Python пакеттерін жасау, орнату және басқаруды үйретеді.  
Модульдер мен ішкі пакеттерді ұйымдастыру, `pyproject.toml` арқылы конфигурациялау және локальды орнату көрсетіледі.

---

## Summary
Жобада қарапайым автокөлікке арналған пакет мысал ретінде көрсетіледі:

- **engine_tools.py**: мотор қуаты, жанармай шығыны, май ауыстыру қажет екенін есептейді.
- **speed_tools.py**: жылдамдыққа қатысты есептерді шешеді, жол уақытын шығарады, ескерту береді.
- **helpers/messages.py**: хабарламаларды форматтау және автокөлік моделін белгілеу.
