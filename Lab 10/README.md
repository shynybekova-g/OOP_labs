# Lab 10 — File Handling and Exception Processing

## Тақырыбы мен мақсаты
Файлдармен жұмыс жасау және қателіктерді өңдеуді меңгеру. Python тілінде try...except конструкциясын қолдану арқылы деректерді қауіпсіз өңдеуге үйрену.

## Тапсырмалар мен кодтар
1. **task1_read_file.py** — Кітаптар тізімін оқу және орташа бағаны есептеу.
2. **task2_write_file.py** — Жаңа кітап қосу және тізімді көрсету.
3. **task3_exceptions.py** — Файлды оқу кезінде қателерді өңдеу.
4. **task4_custom_error.py** — Баға тым төмен болған жағдайда арнайы қате шығару.

## Скриншоттар мен нәтижелер
`screenshots/` папкасында.

## Бақылау сұрақтары
1. try...except не үшін қажет?  
   → Файлдармен жұмыс немесе пайдаланушы енгізу кезінде қателерді өңдеу үшін.

2. Custom Exception қалай жасалады?  
   → `class MyError(Exception): pass` арқылы.

## Қорытынды
